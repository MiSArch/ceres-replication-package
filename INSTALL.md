# CERES usage guideline for setup with MiSArch

1. Go to the `/ceres` directory and unzip the sources including a Docker Compose file for Ceres.
2. Clone all respositories that belong to MiSArch . They can be found under https://github.com/MiSArch.
3. Follow the [installation documentation for MiSArch](https://misarch.github.io/docs/category/installation).
4. Check out the [How to use CERES](https://misarch.github.io/docs/docs/user-manuals/how-to-use-experiment-tool) documentation to use CERES and run experiments.
